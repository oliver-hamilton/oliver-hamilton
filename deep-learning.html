Deep Learning From Scratch

Technologies

  • The library is written entirely in Python.
  • It was developed by me in order to gain a deeper understanding of the implementation of neural networks.
  • Includes implementations of both convolutional and fully connected neural networks, allowing for processing of different types of data, including images.
  • Implements binary cross-entropy, categorical cross-entropy and mean-squared error loss functions, meaning the library can be used for both regression and classification tasks.
  • Features a range activation function modules, including:
    • ReLU
    • Leaky ReLU
    • Linear
    • Logistic (sigmoid)
    • Softmax
