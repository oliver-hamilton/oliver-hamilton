Visual Calculator

Technologies

  • The logic behind this UWP (Universal Windows Platform) application was written in C#, and the visual interface was designed using XAML.
  • It was designed to display fractions, surds and exponents in a natural way that could be easily read by users.
  • The ultimate goal was a user-friendly desktop calculator that could be used in place of a physical scientific calculator.
